【輸入說明】  
第一行為輸入函數類型，輸入 1 代表選擇 f1，輸入 2代表選擇 f2，輸入 0 結束程式  
第二行為輸入四個參數a(函數中的係數)、p(積分區間起始點)、q(積分區間終點)、err(精度要求，表示精確到小數後第error位)  
使用者重複輸入,直到輸入0 結束程式。  
  
範例輸入：  
1 (selected function)  
4 -2 1 9 (a, p, q, err)  
2 (selected function)  
1 0 3 9 (a, p, q, err)  
0 (end)  
【輸出說明】  
輸出根據梯形法計算出的最終結果，答案需輸出到小數點第12位  
若輸入不為1、2、0則輸出Invalid  
範例輸出：  
5.054815608319  
29.752380952687
