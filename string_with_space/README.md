## 利用 fgets 得到含空格的字串  

fgets(字串, 字串大小, 輸入)  

**<注意> fgets 會讀到換行符號，所以透過 strcspn 將換行符號改為字串結束符號**  
(字串長度 - 1，原本的字串結束符號還在，只是多一個在前面)  

ex. Hello World!  
原本應該是  
{'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd', '!', '\\', 'n', '\0'}  
會變成  
{'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd', '!', '\0', 'n', '\0'}
